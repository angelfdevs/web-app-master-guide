### FRONTEND MODELO DDD 
# Desarrollo bajo las tecnologias Vue y JavaScript

Para la creacion del proyecto, primero debemos ubicarnos en la **Terminal**, estar dentro de la carpeta en la cual colocaremos el proyecto y poner el siguiente comando para crear el proyecto.
```bash
npm create vite@latest nombre_del_proyecto -- --template vue
```

Una vez creado el proyecto, nos dirigimos dentro del proyecto mediante un comando
```bash
cd nombre_del_proyecto
```

Luego de ello instalar las librerias
```bash
npm install
```

Y finalmente instalar PrimeVue, Internacionalizacion y Axios
```bash
npm install primevue @primeuix/themes primeflex primeicons axios vue-i18n
```
Antes de programar la logica de negocio, es importante preparar el entorno
